About Number Base Conversion

What are Number Bases?

Different number bases use different symbols to represent values. Decimal (base 10) is familiar to humans, but computers use binary (base 2) internally, hexadecimal (base 16) for memory addresses, and octal (base 8) for file permissions.

Common Bases

  • Binary (2): 0, 1 (computer operations)
  • Octal (8): 0-7 (Unix file permissions)
  • Decimal (10): 0-9 (human standard)
  • Hexadecimal (16): 0-9, A-F (memory addresses, colors)

Real-World Examples

  • Colors: #FF5733 in hex = RGB(255, 87, 51)
  • Permissions: 755 in octal = rwxr-xr-x
  • Memory: 0x7FFF (hexadecimal) = 32767 (decimal)

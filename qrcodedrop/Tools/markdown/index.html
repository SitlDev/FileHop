About Markdown to HTML

What is Markdown?

Markdown is a lightweight markup language for creating formatted text using plain text syntax. Created by John Gruber in 2004, it's used for README files, documentation, blogs, and comment systems across the web.

Markdown Syntax

  • # = Headings (# H1, ## H2, ### H3)
  • **bold** = Bold text
  • *italic* = Italic text
  • [link](url) = Hyperlinks
  • - or * = Unordered lists
  • 1. 2. 3. = Ordered lists
  • ``` = Code blocks

Real-World Use Cases

  • GitHub: READMEs and documentation (default format)
  • Blogs: Static site generators (Jekyll, Hugo, Next.js)
  • Documentation: API docs, technical manuals
  • Comments: Reddit, Discord, Slack, GitHub issues
